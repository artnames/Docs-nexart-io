# NexArt Documentation

NexArt provides verifiable execution infrastructure for AI and deterministic systems.

The platform converts executions into Certified Execution Records (CERs), cryptographically verifiable artifacts that prove exactly what ran. CERs allow independent verification of inputs, runtime state, and outputs without trusting the originating system.

This repository contains the documentation site for the NexArt protocol and execution verification architecture.

## What is NexArt?

AI systems increasingly produce outputs that must be auditable and reproducible. Traditional logs cannot reliably prove what actually ran.

NexArt introduces Certified Execution Records (CERs), tamper-evident artifacts that bind:

- inputs
- parameters
- runtime environment
- outputs

into a single cryptographically verifiable record.

Anyone can later recompute and verify the execution independently.

Learn more:

https://nexart.io
